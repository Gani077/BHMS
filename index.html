Project Overview

Project Title

Battery Health Monitoring System using ESP32 and Data Analytics

Problem Statement

Battery packs used in electric vehicles and energy storage often degrade silently, which can lead to reduced performance, unexpected shutdowns and unsafe operating conditions.

Objective

  • Continuously monitor voltage, current and power of a battery pack.
  • Visualise real-time behaviour and compute key health indicators.
  • Provide a base platform for further data science based prediction.

How the System Works

  • Sensors connected to ESP32 measure voltage, current and temperature.
  • Data is logged and exported as CSV for analysis.
  • This web dashboard loads the CSV, computes analytics and renders charts.

Data Science Integration

The dashboard demonstrates data cleaning, feature extraction and visual analytics. In future, machine learning models can be added for battery health prediction and remaining useful life estimation.

Data Science Workflow

Data Collection from Sensors

Voltage, current and power are captured from the battery pack using appropriate sensing hardware.

Raw Data Logging

Sensor readings are logged over time on a microcontroller or data acquisition system.

Data Cleaning & CSV Conversion

Noise is removed, units are checked and the dataset is exported into a clean CSV file.

Visualization & Analysis

This dashboard loads the CSV, computes analytics and plots charts to understand behaviour.
